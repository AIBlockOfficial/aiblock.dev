## About The Project


The A-Block API allows you to interact with the A-Block chain without the need for complicated, risky smart contracts. It's here to let you work with A-Block in a way that's fast, simple and secure.

The API is used to interact with the three following nodes :

-   Mempool: 
-   Storage
-   Miner
