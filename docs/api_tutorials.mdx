## About The Project


The ABlock API allows you to interact with the ABlock chain without the need for complicated, risky smart contracts. It's here to let you work with ABlock in a way that's fast, simple and secure.

The API is used to interact with the following nodes :

-   [**Mempool**](/docs/network-wiki/mempool-node): responsible for collecting transactions and bundling them into new blocks
-   [**Storage**](/docs/network-wiki/storage-node): responsible for storing the full blockchain history.
-   [**Miner**](/docs/network-wiki/miner-node): responsible for mining new blocks
